    app.services.user_points.queries module¶

    Read-only points queries.

    Aggregations and lookups over user_points for games, tasks and users. These methods never mutate state; the write path lives in app.services.user_points.assignment.

    • enforce_scope (bool) – When True, verify the caller may access the game before returning data.
