    Updates a task parameter row in place by its id.

    Sibling of GameParamsRepository.patch_game_params_by_id(), used by the task PATCH flow to rewrite a param’s key/value without deleting and recreating the row (so its id is preserved).

    Raises NotFoundError if the param does not exist.
